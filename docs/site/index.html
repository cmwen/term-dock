The terminal workspace that stays findable.

Term Dock gives every local or SSH terminal a durable workspace, an attachable session, and a safe path back into work—without turning terminal output into cloud data.

QUICK START

Run locally in three commands.

  1. Install the project dependencies.pnpm install
  2. Launch the desktop application.pnpm tauri dev
  3. Open a workspace and start a terminal.Its definition and session state are saved locally.

Requirements

Node.js 22+, pnpm 11.7+, TypeScript 7.0.2, Rust 1.85+, and the platform prerequisites for Tauri.

DESIGNED FOR REAL WORK

Useful structure, careful boundaries.

01

Durable by default

Workspaces carry their directory, startup command, SSH target, and session identity so you can resume rather than reconstruct.

02

One controller at a time

Only the client with the in-memory controller capability can send input, resize, or terminate a live terminal. Viewers stay read-only.

03

AI-ready, not AI-leaky

Export a versioned, provider-neutral workspace summary. Terminal output, secrets, commands, and telemetry are excluded.

04

Remote-ready architecture

The versioned broker contract supports a future authenticated relay without changing the native trust boundary.
